이온유, 오후 3:40
1. Home 화면
  - 현재 개설된 과목 리스트 확인 가능

------- 과목 클릭 
1. 홈
  - 목차 확인 가능 ( 실습 및 과제 개수 )
  - 좌측 메뉴의 홈, 실습현황, 실습관리만 거의 사용함

2. 실습현황
  - 과목에 할당된 실습 및 과제 리스트 확인 가능
  - 우측의 참여, 제출 쪽 숫자 클릭 시 학생들 제출 여부 확인 가능

3. 실습관리
  - 문제 공개 여부, 제출 기한 등 문제 관리 페이지

------------ 실습 제출 방법
1. 목차 -> 상세보기로 문제 페이지로 이동

2. 우측 하단 실습하기 클릭 ( 강사한테는 '샘플 실행하기' 로 출력됨 )

3. 실습실 입장
  - 좌측엔 gitlab, 우측엔 실행 결과 표시되는 화면이 출력됨
  - 좌측 gitlab 이 출력되지 않으면 gitlab 권한 문제

4. 코드 수정
  - 기본적으로 하나 or 필요하다면 2개의 파일이 생성되어 있음
  - 파일 위의 버튼으로 new file, upload file 등의 기능 사용 가능
  - 파일 클릭 -> 코드 수정 ( vscode 진행 후 복사 붙여넣기 추천 )

5. commit
  - 좌측 하단의 create commit 클릭
  - Commit to master branch 클릭
    -> Commit to new branch 클릭 시 merge request 까지 진행해야 편함
