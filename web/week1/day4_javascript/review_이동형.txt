오늘 가장 기억에 남는 부분
: classList
classList를 통해서 querySelect로 받아온 데이터에 
추가적인 class 명을 붙여서 매핑된 해당 css 특성을 바로 적용할 수 있다는 점

그 외 배운 부분
chrome tab 생성 프로젝트를 진행하면서
오전에 진행하였던 event, key, classList 등의 내용들을
종합적으로 확인할 수 있었다.

event 부분에서는 document.querySelect(".클래스명")를 통해서 
eventListener와 eventHandle의 특성으로 특정 상황 발생 시
어떤 식으로 처리할 지 동작하는 등의 기능 설계가 가능했다.

특정 상황에 대해서는 click, change, keydown, keyup 등이 있었고
각각의 이름들이 뜻하는 바처럼 클릭했을 때, 값이 변경되었을때,
키가 눌렸을 때, 키가 눌렸다 떼어졌을 때 등의 상황을 처리할 수 있었다.

오후에 chrome tab 생성 프로젝트에서
flex를 이용한 정렬을 시작으로 container와 wrapper 특성을 모두 활용해서
각 list들이나 div, image들을 한꺼번에 정렬해주었고, 클릭 이벤트 시 
다크모드 전환 효과도 적용할 수 있었다.

끝으로 
location.href = "url+검색내용" // 바로 가기
window.open("url+검색내용") // 새 탭 열기
이 두 가지 방식으로 원하는 검색내용을 구글 검색으로 찾는 기능을 학습했다.

이러한 부분에 있어서 특정 쿼리를 날려서 원하는 페이지로 가서
원하는 내용을 입력하는 방식을 자동으로 단시간에 전달하여
티케팅 봇을 개발할 수도 있을 것 같다는 생각을 하기도 했다.
